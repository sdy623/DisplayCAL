Info
====

Name:        RealDisplaySizeMM
Platform:    Linux / Mac OS X / Windows
Description: Get the size (in mm) of a given display.
             Borrowed from Argyll CMS (http://argyllcms.com).

Building & Installing
=====================

python setup.py install

Usage
=====

import RealDisplaySizeMM

RealDisplaySizeMM.RealDisplaySizeMM(0) # get size of 1st display
RealDisplaySizeMM.RealDisplaySizeMM(1) # get size of 2nd display
